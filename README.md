# RENGE
RENGE infers gene regulatory networks (GRNs) from time-series single-cell CRISPR screening data.

## Requirements
```
numpy == 1.19.5  
pandas == 1.1.5  
scipy == 1.4.1  
scikit-learn == 0.21.3  
statsmodels == 0.10.1  
jax == 0.2.0  
optuna == 2.3.0  
```

## Download
```
git clone https://github.com/masastat/RENGE
cd RENGE
```
